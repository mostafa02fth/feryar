# Feryar

A client-side web app for learning languages from video, built with React and TypeScript.

> Status: early stage, work in progress. Built in public.

## Tech stack

- Vite + React 19 + TypeScript
- Tailwind CSS v4 + shadcn/ui
- Vidstack (media playback)
- ffmpeg.wasm (in-browser media processing)

## Getting started

```bash
pnpm install
pnpm dev
```

## About the name

Feryar comes from Kurdish: fêr (to learn) + yar (companion) — a learning companion.
